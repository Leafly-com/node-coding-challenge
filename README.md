# node-coding-challenge

Welcome to the Leafly Node Coding Challenge!

This project uses `jest` as a test runner. To install dependencies, run: 

`npm i`

Select the file with the exercise specified during the interview located in the `src` directory. There are instructions, expectations, and examples within each file specific to the exercise. 

To run the assertions tied to the specific exercise, you can run the test command appending the relevant path. For example: 

`npm test src/find-sums.test.js`
